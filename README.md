# Installation
**you need to have these fonts/programs in order this dotfile works**
- rofi
- ft-lab picom **only** (you can install this fork of picom in https://github.com/r0-zero/picom)
- polybar
- alacritty (or any other prefered teminal just set the font to hack and opacity 0)
- hack font and its symbols

the installation is very simple you just put background.png in your home directory and copy paste the configs directrory in the .config folder
if you want to use anyother terminal thats not alacritty you should make a config that has opacity = 0 and uses hack font and also change **$terminal** in the autostart (aka config of herbstluftwm)
# keybinds 
* <kbd>Super</kbd> + <kbd>Space</kbd> — Open app launcher (`rofi`)
* <kbd>Super</kbd> + <kbd>Enter</kbd> — Open terminal (`alacritty`)
* <kbd>Super</kbd> + <kbd>f</kbd> — Toggle floating mode
* <kbd>Super</kbd> + <kbd>Tab</kbd> — Cycle frame / grid layouts
